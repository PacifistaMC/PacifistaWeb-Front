Wiki de Pacifista

Bienvenue sur ce wiki, dans ce wiki vous allez en savoir plus sur le fonctionnement de Pacifista avec ses commandes et ses fonctionalitées. Pour commencer je vous conseille de lire les règles du serveur disponibles en cliquant ici si vous ne les connaissez pas encore.

Pacifista est un serveur survie libre, vous arrivez, vous pouvez faire votre maison/base où vous voulez (pas trop proche d'un autre joueur sauf si il vous autorise), miner bref faire une survie classique mais tout en étant sur un serveur.

À l'heure actuelle Pacifista compte deux serveurs survie :
- Serveur Survie
- Serveur FreeBuild

Voici les différences entre les deux serveurs :
Serveur Survie
- PvP Activé
- Perte d'inventaire lors de la mort (Inventaire sécurisé les joueurs ne peuvent pas prendre votre inventaire)
- Explosion de creepers activé

Attention : Les inventaires, EnderChest, Armures, Barres d'exp et monnaies sont synchronisés entre les deux serveurs.

Quand vous vous connectez sur Pacifista vous allez arriver sur un serveur d'accueil nommé Hub, pour choisir votre serveur de jeu vous avez une boussole dans votre inventaire pour ouvrir le menu de sélection de serveur. Vous pouvez aussi accéder à ce menu depuis n'importe quel serveur avec la commande /menu qui vous ouvre le menu principal de Pacifista.

Dans le /menu vous allez avoir différents boutons :
- La boussole : Ouvre le menu de sélection de serveurs
- L'oeuf : Ouvre le menu d'achat d'oeuf -> Disponible sur les serveurs Survie et FreeBuild
- Le minecart coffre : Pas encore disponible

Vous savez maintenant les bases du serveur. Pour en savoir plus sur les autres fonctionnalités n'hésitez pas à aller voir les autres wikis disponibles.
